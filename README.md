# IdleGameModule
방치형 게임에서 자주 사용 되는 라이브러리들을 쉽게 사용할 수 있게 하는 모듈 


# Requirements
- UniTask (v2.4.1)
- TheBackend (v5.11.1)
- Google Login
- Apple Auth Service
- IronSource (v)
- iOS Advertising Support (1.0.0)
- BigDouble (BreakInfinity)


## 프로젝트 세팅

### 뒤끝
- SendQueue를 전부 이용하기 때문에 SendQueueMgr 스크립트를 포함한 오브젝트를 게임 시작 씬에 띄워준다.
